## 🚀 দ্রুত শুরু (৫ মিনিট)

### Step 1: Supabase সেটআপ
```
1. supabase.com এ যান
2. Sign Up করুন
3. নতুন প্রজেক্ট তৈরি করুন
```

### Step 2: ডাটাবেস তৈরি
```
Supabase → SQL Editor → DATABASE_SETUP.sql রান করুন
```

### Step 3: Config আপডেট
```javascript
// assets/js/config.js
const SUPABASE_CONFIG = {
    URL: 'আপনার-URL',
    ANON_KEY: 'আপনার-KEY'
};
```

### Step 4: ডিপ্লয় করুন
```bash
vercel deploy
# অথবা GitHub এ পুশ করুন
```

## 🐛 সমস্যা সমাধান

### ❌ Logout বাটন কাজ না করলে?
```
সমাধান:
1. Browser Console খুলুন (F12)
2. কী এরর দেখাচ্ছে চেক করুন
3. supabase-client.js এ logout() ফাংশন চেক করুন
```

### ❌ ডেটা সেভ না হলে?
```
সমাধান:
1. Supabase URL এবং KEY সঠিক কিনা চেক করুন
2. DATABASE_SETUP.sql আবার রান করুন
3. Supabase ড্যাশবোর্ডে 'about' টেবিল আছে কিনা চেক করুন
```

### ❌ স্টাইল প্রয়োগ না হলে?
```
সমাধান:
1. Cache পরিষ্কার করুন (Ctrl+Shift+R)
2. style.css লোড হয়েছে কিনা চেক করুন (F12 → Network)
3. CSS পাথ সঠিক কিনা নিশ্চিত করুন
```

## 📊 ডিপ্লয়মেন্ট অপশন

### Vercel (সবচেয়ে সহজ - সুপারিশকৃত) ⭐
```bash
npm install -g vercel
vercel

```

### Netlify
```
1. netlify.com এ যান
2. GitHub অ্যাকাউন্ট কানেক্ট করুন
3. রিপোজিটরি নির্বাচন করুন
4. ডিপ্লয় ক্লিক করুন
```

### GitHub Pages
```
1. নতুন রিপোজিটরি তৈরি করুন: username.github.io
2. ফাইল পুশ করুন
3. Settings → Pages → Enable করুন
4. বস! লাইভ!
```
